============================================================
WHAT ATLASCHAT IS
============================================================
AtlasChat is a governance-first browser extension that enforces
safe, deliberate, and visible interaction with AI systems.

AtlasChat operates at the user interface boundary and governs:
- Prompt handling modes
- User intent signaling
- Minimal state persistence
- Governance visibility

AtlasChat does not replace AI models or platforms.

============================================================
WHAT ATLASCHAT IS NOT
============================================================
AtlasChat is NOT:
- An AI model
- A chatbot competitor
- A secure enclave
- A compliance accreditation
- A backend service
- A data storage system

============================================================
CORE FEATURES
============================================================
Prompt Modes:
- Guard Mode: default safe mode, no execution
- Direct Mode: explicit execution, no rewrite

State Scope:
- NONE
- LIGHT (metadata only)

============================================================
PRIVACY & DATA HANDLING
============================================================
AtlasChat v1.0 does NOT:
- Store prompts
- Store responses
- Store conversations
- Transmit data externally

Only minimal metadata is stored locally for governance visibility.

============================================================

============================================================
INSTALLATION
============================================================
1. Extract the release package
2. Load the extension as unpacked in Chrome/Edge
3. Enable AtlasChat via the toolbar icon
4. Select desired mode
