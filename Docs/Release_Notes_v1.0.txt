==============================

Release Date:
2025-12-16

Status:
CERTIFIED • FROZEN • WORKING BASELINE

----------------------------------------------------------------
OVERVIEW
----------------------------------------------------------------
AtlasChat v1.0 is a governance-first browser extension designed to help users
interact with powerful AI systems deliberately, visibly, and responsibly.

AtlasChat does NOT replace AI models or platforms.
It governs how users interact with them at the UI boundary.

This release establishes a hardened, auditable baseline focused on safety,
clarity, and user intent — without backend services or data storage.

----------------------------------------------------------------
CORE FEATURES
----------------------------------------------------------------

Prompt Governance Modes
-----------------------
• PME (Prompt Mode Enhanced) — Default
  - Clarifies and rewrites user intent
  - Produces execution-ready prompts
  - Does NOT execute requests

• PMD (Prompt Mode Direct)
  - Sends user prompts exactly as written
  - Explicit user intent required
  - No rewriting, no interference

Modes are mutually exclusive and always visible.

Domain Modes
------------
• General
• Engineering
• Research

Domains influence rigor, structure, and response expectations.

State Scope
-----------
• NONE
  - No persistence

• LIGHT (Default)
  - Metadata only:
    - Last mode
    - Last domain
    - Timestamps
    - Enforcement events

AtlasChat v1.0 does NOT store:
• Prompt content
• Response content
• Conversation history

----------------------------------------------------------------
ARCHITECTURE SUMMARY
----------------------------------------------------------------
• Browser extension (Chrome / Edge)
• Manifest v3
• No backend services
• No external libraries
• Governance enforced before prompts leave the browser
• Fail-open safety model (user intent always wins)

----------------------------------------------------------------
WHAT ATLASCHAT IS NOT
----------------------------------------------------------------
• Not an AI model
• Not a chatbot competitor
• Not a compliance accreditation or ATO
• Not a secure storage or enclave
• Not a replacement for approved systems

AtlasChat governs user behavior — not vendor infrastructure.
